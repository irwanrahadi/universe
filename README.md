# Isra Mikraj & Fenomena Ruang-Waktu dalam Al-Qur'an

Kajian tematik interaktif: ayat-ayat paralel tentang waktu, analisis matematis rasio antar-alam (1:1.000 dan 1:50.000), sumbu kosmik Ka'bah–Baitul Ma'mur, dimensi interior (lathā'if & Martabat Tujuh), analisis lafadz QS 27:40, taksonomi 13 register ayat waktu, resonansi ilmu komputer & komputasi kuantum — dengan **protokol pelabelan epistemik** pada setiap klaim (nash qath'i → hadits → atsar → isyārī → ijtihad → ilustrasi fisika → spekulatif → ditolak).

## Isi

| Berkas | Keterangan |
|---|---|
| `index.html` | Situs lengkap satu halaman: seluruh bagian kajian + **Simulator Frame Waktu** + **Visualisasi 3D heliks kerucut** (Three.js via CDN) |
| `dokumen/Isra_Mikraj_Ruang_Waktu_AlQuran_v12.docx` | Dokumen lengkap 17 bagian (versi Word, landscape) |

## Cara deploy di GitHub Pages

1. Buat repositori baru di GitHub (mis. `kajian-isra-mikraj`).
2. Unggah seluruh isi folder ini ke repositori (drag-and-drop di web GitHub juga bisa).
3. Buka **Settings → Pages**.
4. Pada *Source*, pilih **Deploy from a branch**, branch `main`, folder `/ (root)`, lalu **Save**.
5. Tunggu ±1 menit; situs tayang di `https://<username>.github.io/kajian-isra-mikraj/`.

Tidak ada proses build — murni HTML/CSS/JS statis. Three.js dimuat dari CDN (cdnjs), sehingga butuh koneksi internet saat halaman dibuka.

## Fitur interaktif

- **Simulator Frame Waktu** — satu jam acuan berjalan; tujuh odometer (frame manusia, Lailatul Qadr, Kahfi, 1:1.000, Bilqis, 1:50.000, kiamat-satu-shalat) berlomba pada lajunya masing-masing. Kontrol: mulai/jeda/reset, laju 1×/10×/60×.
- **Heliks 3D (tamsil geometris)** — perumpamaan visual atas pola bilangan: spiral kerucut dengan dua proyeksi hidup: dari atas = spiral melingkar; dari samping = gelombang sinus teredam. Seret untuk memutar, scroll/cubit untuk zoom.

## Catatan metodologis

Kajian ini adalah **kerangka interpretif**, bukan teori ilmiah — tidak ada klaim prediksi falsifiable. Seluruh perhitungan fisika berstatus ilustrasi; seluruh pembacaan simbolik berlabel isyārī. Rincian protokol pelabelan ada di bagian penutup situs dan Bagian 17 dokumen.
